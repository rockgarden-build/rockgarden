Ideas and potential features for future versions.

## Planned

- **Search** - Client-side search index generation
- **Backlinks** - Show pages that link to the current page
- **Custom themes** - Theme loading from site directory
- **Watch mode** - Rebuild on file changes

## Maybe

- **Graph view** - Visual link map (like Obsidian)
- **Tags page** - Auto-generated tag index
- **RSS feed** - Feed generation for blogs
- **Syntax highlighting** - Code block highlighting
- **Table of contents** - Per-page heading outline
- **Dataview-lite** - Simple query blocks
- **PDF export** - Single-file output

## Non-Goals

- JavaScript-heavy features
- Database or server requirements
- Breaking Obsidian vault compatibility
- Complex plugin systems

## Contributing

This project is open source. Contributions welcome via GitHub issues and pull requests.
