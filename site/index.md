---
title: Rockgarden
---

A Python static site generator for Obsidian vaults.

## Features

- Works with Obsidian vaults as-is
- Wiki-link resolution (`[[Page Name]]`)
- Embeds (`![[image.png]]`)
- Callout support
- Customizable templates and themes

## Quick Start

```bash
# Install
uv add rockgarden

# Build your vault
rockgarden build --source ./my-vault --output ./_site
```
