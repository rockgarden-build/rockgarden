# Text Formatting

Standard markdown formatting works in Obsidian vaults.

## Emphasis

- *italic* or _italic_
- **bold** or __bold__
- ***bold italic***
- ~~strikethrough~~

## Lists

Unordered:
- Item one
- Item two
  - Nested item

Ordered:
1. First
2. Second
3. Third

Task list:
- [x] Completed task
- [ ] Pending task

## Code

Inline: `const x = 42`

Block:
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Blockquotes

> This is a blockquote.
> It can span multiple lines.
