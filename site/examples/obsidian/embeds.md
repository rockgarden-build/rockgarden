## Syntax Reference

| Syntax | Result |
|--------|--------|
| `![[image.png]]` | Basic embed, filename as alt |
| `![[image.png\|description]]` | Custom alt text |
| `![[image.png\|100]]` | Width of 100px |
| `![[image.png\|100x200]]` | Width 100px, height 200px |
